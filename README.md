# How to run the project?
1. Create venv
2. Activate venv 
    - On macOS/Linux run:
   ``source ./.venv/bin/activate``
    - On Windows run: 
    ``./.venv/Scripts/Activate``
3. Run this command:
   ``pip install -r requirements.txt``
4. Than run ``python snake_release.py`` and here you go!
